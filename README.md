### 分销系统的核心。    
 
| |一级|二级|
|:----:|:----|----:|
|价格|20W|10W|
|一级提成|10W|5W|
|二级提成|5W|2.5|
|维护费|2W|1W|
 
 注：    
 一级可以发展下线，二级不可以。
 提成只是针对一级的。
 
 为什么只有两级，这涉及到法律问题。
 软件并不会对级别做限制。
